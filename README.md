# EpiPy
A Tool for fitting epidemic models.

## Requirements
 * Python 2.7.9
 * PyQt 4
 * [PyQtGraph][2] 0.9.10

## Installation
`$ git clone git@github.com:ckaus/EpiPy.git`

## Start EpiPy
`$ python src/main.py`

## Documentation
* Project [wiki][1]
* Source by using [Sphinx][3]:
  * `$ cd doc`
  * `$ make html`
